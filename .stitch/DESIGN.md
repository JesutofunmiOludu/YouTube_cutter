# Design System for VidMind AI

## 🎨 Brand Identity
- **Name**: VidMind AI
- **Logo**: [V] VidMind
- **Vibe**: Smart, Helpful, Modern, Professional, Innovative, Academic

## 🎨 Color System
- **Primary**: Brand Blue (#3B82F6) - Used for primary CTAs and active states.
- **Gradient**: Blue (#3B82F6) to Purple (#8B5CF6) - For premium features and hero sections.
- **Background**: Gray 900 (#111827) - Dark mode by default for learning environments.
- **Surface**: Gray 800 (#1F2937) for cards and main content.
- **Text**: Gray 50 (#F9FAFB) for primary typography, Gray 300 (#D1D5DB) for secondary.

## 📝 Typography
- **Primary**: Inter (sans-serif)
- **Technical**: JetBrains Mono (monospace for timestamps/code)

## 📐 Layout & Components
- **Border Radius**: 
  - Buttons: 8px (radius-lg)
  - Cards: 12px (radius-xl)
- **Grid**: 12-column grid, max-width 1280px.
- **Shadows**: Soft shadows (shadow-sm) with hover elevation (shadow-md).

## 💡 Design System Notes for Stitch Generation
When generating screens for VidMind AI, strictly adhere to the following:
- Use **Inter** as the primary font and **JetBrains Mono** only for technical data.
- Use **#3B82F6** for all primary action buttons.
- Apply **12px rounded corners** to all cards and **8px** to buttons.
- Keep layouts clean with generous whitespace (4px base unit).
- Use **Gray 900 (#111827)** for all page backgrounds and **Gray 800 (#1F2937)** for elements and cards (Dark mode). Use **Gray 50 (#F9FAFB)** for text.

A comprehensive User Settings and Preferences page for VidMind AI.

**DESIGN SYSTEM (REQUIRED):**
When generating screens for VidMind AI, strictly adhere to the following:
- Use **Inter** as the primary font and **JetBrains Mono** only for technical data.
- Use **#3B82F6** for all primary action buttons.
- Apply **12px rounded corners** to all cards and **8px** to buttons.
- Keep layouts clean with generous whitespace (4px base unit).
- Use **Gray 900 (#111827)** for all page backgrounds and **Gray 800 (#1F2937)** for elements and cards (Dark mode). Use **Gray 50 (#F9FAFB)** for text.

**Page Structure:**
1. **Sidebar Navigation (Internal)**: Profile, Account Security, Notifications, Billing & Subscription, AI Personalization, API Keys.
2. **Profile Section**: Avatar upload, Name, Email, Bio.
3. **Account Security**: Password change fields, Two-factor authentication toggle.
4. **Billing & Subscription**: Current plan summary, Billing history list, "Manage Subscription" button.
5. **AI Personalization**: Preferred research depth (Concise, Balanced, Detailed), Default citation style.
6. **API Keys**: Manage keys for integrations (displayed in JetBrains Mono).
7. **Action Bar**: "Save Changes" (#3B82F6 primary) and "Discard" buttons.
